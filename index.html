Para usar disponibilizar bibliotecas no repositório siga os passos abaixo:

  1. Faça checkout do repositório:
    		git clone -b gh-pages git@github.com:dextra/dextra-maven-repo.git
  2. Faça maven deploy-file para a pasta do repositório:
    		mvn deploy:deploy-file -DgroupId=SK.morena -DartifactId=morena -Dversion=6.4.1 -Dpackaging=jar -Dfile=/projects/shared/confidence_operacao/m2_repo/SK/morena/morena/6.4.1/morena-6.4.1.jar -Durl=file:///tmp/dextra-maven-repo/repository/ -DpomFile=/projects/shared/confidence_operacao/m2_repo/SK/morena/morena/6.4.1/morena-6.4.1.pom
  3. Execute o script:
    		./update-directory-index.sh
  4. Faça commit:
    		git add -A
		git commit -m "..."
		git push origin gh-pages
  5. Configure no pom.xml do seu projeto:
                    <repository>
                        <id>static</id>
                        <url>http://dextra.github.com/dextra-maven-repo/repository/</url>
                        <snapshots>
                                <enabled>false</enabled>
                        </snapshots>
                        <releases>
                                <updatePolicy>never</updatePolicy>
                        </releases>
                </repository>

    Obs: lembre-se de configurar o proxy no seu settings.xml
